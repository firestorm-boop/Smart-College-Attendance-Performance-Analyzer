# Smart-College-Attendance-Performance-Analyzer
C program for college attendance tracking. Inputs student data, computes averages/grades, flags low attendance, shows topper. Built for CS coursework.
## Hostel Mess Billing Tracker
C system for 30-day attendance/billing (₹50/day, 10% discount >5 leaves). Flags unpaid, revenue summary.

Sample: 20 students, mixed attendance/payments.
## Crypto Trading Strategy (Chronos)
